# Ecommerce Project

Online grocery delivery website

	simple shopping website using Flask.

- Dependencies

	Python3
	
	Flask
	
	MySQL

- How to run

	Set up database by running db.sql
	
	Install venv using requirements.txt
	
	Run project
	
- Commands

	$ pip install -r requirements.txt

	$ run wsgi.py

- Enter localhost:5000 in the browser.
